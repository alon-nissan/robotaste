# Archive Directory

This directory contains archived code and logs for reference only. Files here are not actively used in the application.

## Structure

### `legacy/`
Replaced or obsolete code that has been superseded by newer implementations. Kept for historical reference.

### `dev/`
Development and debugging scripts used during development. These are not part of the production codebase but may be useful for hardware debugging.

### `logs/`
Old session logs from development and testing.

## Important Notes
- **Do not import** from this directory in active code
- Files here are tracked in git for historical reference
- For active development, refer to the main `robotaste/` directory
